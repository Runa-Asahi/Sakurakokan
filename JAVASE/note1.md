#  **Javase Note1**
1. win+r -->cmd
2. 默认操作的是C:Users\moon(admin)
3. 常见CMD命令：
   * 盘符名称+冒号：eg.E:切换E盘
   * dir：说明根目录下的所有内容
   * cd 目录
   * cd.. 回退
   * cd 目录1\目录2\...
   * cd \回退到盘符的根目录
   * cls 清屏
   * exit 关闭
4. 常用cmd命令
   * qq.exe 在当前路径里打开qq
   * 把qq的路径记录在电脑的环境变量
  * 配置环境变量：1.打开我的电脑系统设置页面，属性，高级系统设置，高级，环境变量，系统变量path，编辑，新建，把完整路径放进去，上移
5. 步骤：用记事本编写程序 编译文件 运行文件
   1. 切换盘符：要先找到我们刚刚书写的代码
   2. 编译：javac是JDK提供的编译工具，我们可以通过这个工具把当前路径下的java文件编译成class文件
   3. 运行：java也是JDK提供的一个工具，作用就是用来运行代码的，在运行的时候是不加后缀名的
