## 算法竞赛阶段性总结

#### STL相关内容
1. 统一使用size()方法获取容器元素个数
2. 统一使用empty()方法判断容器是否为空
3. clear的行为是移除容器中的所有元素，
   除了array,stack,queue,priority_queue均可使用clear
4. Stack相关内容
   1. stack的初始化
   ```cpp
   stack<int> stk;//创建一个空栈
   //栈 不允许初始化 但允许 从已有的栈进行拷贝构造
   stack<int> stk2(stk);
   stack<int> stk3 = stk2;
   ```
   2. 入栈:push(); 出栈:pop(); 取出栈顶元素(取出元素 但不会被pop()掉):top();
   3. 清空栈:stack中不能遍历
   ```cpp
   while(!stack.empty()) stack.pop();//时间复杂度O(n)
   stack = {};//时间复杂度O(1)
   ```
5. queue相关内容
   1. queue的初始化与stack相同
   2. 入队:push(); 出队:pop(); 取出队首元素:front(); 取出队尾元素:back();

#### 排序：
1. 快速排序 
   时间复杂度为O(nlogn),一般在数据范围1e6以内使用快排,且元素的大小一般很大
   ```cpp
   sort(v.begin(),v.end());//vector中
   sort(a,a+n);//C数组中
   ```
2. 桶排序
   时间复杂度为O(n),当元素的大小比较小的时候使用,利用值域小的特点
   桶排序 必须有 桶编号
   ```cpp
   int a[N];
   for(auto x:arr){
        a[x]++;
   }
   for(int i=1;i<=n;i++){
        for(int j=1;j<=a[i];j++){
            cout<<i<<" ";
        }
   }
   ```
